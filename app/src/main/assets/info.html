연가

■ 연가 일수

  • 1년차: 15일 (이월 불가)
  • 2년차: 13일
  • 병가 미사용 시: +5일
  • 병가 2일 이하 사용 시: +2일
  • 병가 3일 이상 사용 시: 추가 연가 없음

■ 연가 사용 규정

  • 1일 또는 반일(4시간) 단위로 사용 가능
  • 지각, 조퇴, 외출 등은 10분 단위로 사용 가능
  • 오전 반가: 09:00~14:00 / 오후 반가: 14:00~18:00

■ 연가 이월

  • 복학 사유 외 이월 불가
  • 이월 시 연속 사용만 가능하며 이후 연가 사용 불가

■ 연가 자동 소멸

  • 사용하지 않으면 자동 소멸
  • 연가보상제도는 적용되지 않음
  • 연가 소멸 4개월 전부터 제한 불가하다는 내부 규정 있었으나, 병무청 공식 답변은 "없다"

■ 2년차 연가 당겨쓰기

  • 기관장 허가 시 최대 7일까지 당겨쓰기 가능

■ 연가 승인 및 거부

  • 기관장이 승인
  • 업무 공백, 동일 시기 집중 사용 시 조정 가능
  • 연가 거부 시 공문으로 사유 통보 의무

■ 근무지에 따른 제한

  • 초기 3개월 제한 관행 있음
  • 학교는 방학 기간 사용 권장
  • 업무량 많은 기관은 사전 협의 필수

■ 마지막 근무일

  • 소집해제 당일은 연가/병가 사용 제한 가능
  • 설문 후 부분 연가로 조기 퇴근 가능

■ 특별휴가/기타

  • 기관 휴일은 원칙상 연가 아님 (특별휴가 권고)
  • 근로자의 날, 창립기념일 등도 연가 대상 아님

■ 허가지각 / 조퇴 / 외출

  • 사전 또는 사후 허가 필요
  • 연가 또는 병가에서 시간 공제
  • 허가 없이 진행 시 무단 처리, 복무기간 연장 가능

■ 기타

  • 연가 신청은 사회복무포털에서도 가능하나 실사용 시 기관 통보 필요
  • 복무상황부 미기록 시 감사 대상
