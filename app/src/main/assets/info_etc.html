앱 정보

  • 앱 이름 : 얼마나 쉴 수 있지?
  • 제작 : MINGYU OH

■ 주요 기능

  • 입대일 입력만으로 자동 전역일 계산
  • 1년차 연가, 2년차 연가, 병가 관리
  • 1ms 단위 D-day & 진행률 표시
  • 진행 상황 자동 저장
  • 다크모드 대응
  • 전체 초기화 기능
  • 휴가와 관련한 여러 정보 제공
    - 사실과 다르거나 현재 달라진 내용이 있을 수 있습니다. 애매한 부분은 꼭 담당 복무지도관에게 물어보세요!
    - 모든 불이익에 대해서 개발자는 책임을 지지 않습니다
  • 간결하고 직관적인 인터페이스

■ 사용 방법

  1. 초기 화면에서 입영일 선택하여 본인의 입영일을 선택합니다
  2. 사회복무 포털에 접속하여 본인이 사용한 연,병가일을 확인합니다
  3. 사용한 일, 시간, 분 단위로 각각 칸에 맞게 입력하여 오른쪽의 체크 버튼을 누르면 자동으로 차감됩니다.
  4. 실수로 잘못 차감 시켰을 경우 메인화면 상단 빨간 RST 버튼으로 초기화할 수 있습니다.
    !! 저장된 모든 연/병가가 초기화됩니다.
  5. 초기화 메뉴에 들어가시면 앱 전체 초기화와 전역일 계산 초기화도 가능합니다.

■ 개인정보 및 보안

  • 개인정보를 수집하지 않습니다. 이 앱은 인터넷에 연결되지 않으며 기기의 어떠한 권한도 요청하지 않고, 어떤 형태의 사용자 정보도 외부로 전송하지 않습니다.
  • 모든 데이터는 기기 내에만 저장됩니다. 입력한 정보는 오직 사용자 본인의 Android 기기에만 저장되며, 다른 앱에서 접근할 수 없습니다.

■ 제작 의도

사회복무포털에서 사용한 연/병가만 표시되고 얼마나 남아있는지 직관적으로 확인이 안되는 문제와
확인 가능한 다른 좋은 앱이 있지만, 저에게는 그 앱의 모든 기능이 필요가 없고 광고가 있어서 직접 만들었습니다.
